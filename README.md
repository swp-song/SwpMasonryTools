# SwpMasonryTools



##### 导入：

```
手动导入：

SwpMasonryTools 文件夹 导入 项目 中, 依赖  Masonry 自动布局框架
#import "SwpMasonryTools.h"
```
---

```ruby

CocoaPods 导入:
使用  CocoaPods 会自动 pod Masonry
pod search SwpMasonryTools

pod 'SwpMasonryTools'

#import <SwpMasonryTools/SwpMasonryTools.h>

```
